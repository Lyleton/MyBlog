## 关于这个博客

欢迎来到我的技术博客！这里是我分享技术学习心得、开发经验和思考的地方。

### 博客主题

这个博客主要涵盖以下内容：

- **前端开发** - Vue.js、React、TypeScript 等现代前端技术
- **后端开发** - Node.js、数据库、API 设计
- **DevOps** - Docker、CI/CD、云服务部署
- **编程思想** - 设计模式、最佳实践、代码质量

### 技术栈

本博客使用以下技术构建：

- **框架**: Nuxt 3 (Vue 3 + Vite)
- **样式**: Tailwind CSS + CSS Variables
- **内容**: @nuxt/content (Markdown + MDC)
- **部署**: 静态生成 (SSG)
