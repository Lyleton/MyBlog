# Hello World

欢迎来到我的技术博客！这是基于 **Nuxt 3** 和 **@nuxt/content** 构建的个人博客系统。

## 技术栈

本博客使用以下技术构建：

- **框架**: Nuxt 3 + Vue 3
- **内容管理**: @nuxt/content
- **样式**: Tailwind CSS
- **部署**: Vercel

## 特色功能

### 终端风格 UI

整个博客采用程序员熟悉的终端/代码编辑器风格设计：

```bash
$ cd ~/blog
$ ls ./articles
hello-world.md
vue-composition-api.md
typescript-tips.md
```

### 代码高亮

支持多种语言的代码高亮：

```typescript
const greeting = (name: string): string => {
  return `Hello, ${name}!`
}

console.log(greeting('World'))
```

### 暗色模式

支持亮色和暗色主题切换，自动适应系统偏好。
